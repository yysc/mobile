Examples and documentation for using the IDX One UI enablement for mobile platforms in applications.

The following features are available in this directory:

  • mobile.html — getting started with using the IDX One UI enablement for mobile platforms.
  • galleryviewer.html — a widget gallery and theme tester for the One UI mobile theme.
  • gallery.html — a widget gallery and theme tester for the One UI mobile theme (full browser mode).
  • header.html — overview, primer and demo for the IDX support for the One UI "header" standard.
  • launch.html — overview, primer and demo for the IDX support for the One UI "launch" standard.
  • login.html — overview, primer and demo for the IDX support for the One UI "login" standard.
  • about.html — overview, primer and demo for the IDX support for the One UI "about" standard.
