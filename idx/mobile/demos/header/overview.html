The One UI IDX toolkit provides assistance with complying with the One UI mobile header standards for mobile application development.

At present, no custom widgets are needed to comply with the standard. Using regular dojox.mobile widgets in combination with the mobile themes supplied in the One UI IDX toolkit should normally be sufficient. In a few cases Javascript code will also be required to implement some features.

The dojox.mobile widgets that are normally used to implement the features described in the header standard are:

  • dojox/mobile/EdgeToEdgeList
  • dojox/mobile/Heading
  • dojox/mobile/ListItem
  • dojox/mobile/TabBar
  • dojox/mobile/TabBarButton
  • dojox/mobile/ToolBarButton
  • dojox/mobile/Tooltip
